Splash Page

Technical Description

The Splash Page serves as an introductory screen, utilizing HTML5 for structure and CSS3 for styling. A key feature is the <meta http-equiv="refresh" content="4; url=home.html"> tag, which redirects users to the home page after 4 seconds. The page features a full-screen background video implemented with the <video> tag (attributes: autoplay, muted, loop) for an immersive experience. CSS properties like position: absolute and object-fit: cover ensure the video spans the viewport. A circular loader animation was added using CSS @keyframes to indicate loading progress, enhancing user engagement during the redirect.

Accessibility

Accessibility was addressed by including alt attributes for the logo image (alt="Life Below Water splash screen logo") and fallback text for the video (Your browser does not support the video tag). High-contrast text (white on a dark background) ensures readability, and the tabindex="0" attribute was applied to key elements for keyboard navigation.

Volunteer Page

Technical Description

The Volunteer Page was built with a responsive layout using flexbox (display: flex; flex-wrap: wrap) for the card container, ensuring adaptability across devices. The hero section uses a background image with an overlay styled via position: relative and background-color: rgba for text readability. Volunteer opportunity cards leverage <img> tags with object-fit: cover for consistent image sizing, and buttons feature :hover effects (e.g., background-color transitions) for interactivity. A dropdown menu (<select>) enhances usability by allowing sorting of opportunities.

Accessibility

Each card image includes descriptive alt text (e.g., alt="Sea Turtle Conservation"), and the navigation bar supports keyboard focus with tabindex. High contrast ratios were maintained (e.g., white text on dark buttons), and semantic HTML (<section>, <nav>) improves screen reader compatibility.

Content Page (content_ST1.html)

Technical Description

The Content Page uses semantic HTML (<main>, <section>) to structure information on sustainable fishing. Internal navigation is facilitated by anchor links (<a href="#principles">) and a fixed "Back to Top" button styled with position: fixed and border-radius. Images are responsive (width: 90% from CSS) and paired with text in a flex-based layout (.content-container). The overlay on the main image uses position: absolute and background-color: rgba for a modern design effect.

Accessibility

Images include alt text (e.g., alt="Content Image"), and the "Back to Top" button has an aria-label="Return to top of page" for screen readers. A logical heading structure (<h1> to <h2>) aids navigation, and text contrast meets WCAG guidelines.

Challenges and Lessons Learned

A challenge with the Splash Page was ensuring consistent redirect timing across browsers, resolved by testing and adjusting the meta refresh value. On the Volunteer Page, aligning cards responsively required tweaking flexbox properties (e.g., flex-basis). The Content Pageâ€™s fixed button initially overlapped content, fixed by adjusting z-index. These experiences reinforced the importance of cross-browser testing and early validation for cleaner code.

Compliance

All pages adhere to HTML5 and CSS3 standards, validated via W3C tools. Accessibility aligns with WCAG 2.1 guidelines (e.g., contrast, keyboard navigation). Content and images are original or royalty-free, with proper attribution, complying with web publication regulations like copyright and accessibility mandates.

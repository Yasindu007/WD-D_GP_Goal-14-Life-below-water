Yasindu De Silva, Splash & Volunteer Developer

Splash Page

Technical Description

I used a meta tag (<meta http-equiv="refresh" content="4;url=home.html">) to redirect to the home page after 4 seconds. The loader animation was created using CSS with a spinning border effect (border-top and animation: spin). The background image of the ocean enhances the marine theme, styled with background: url().

Accessibility

The page includes semantic HTML (e.g., <header>, <main>) and a clear color contrast between text and background for readability.

Volunteer Page

Technical Description

The Volunteer Page uses a flexbox layout for the program cards (display: flex; flex-wrap: wrap;). The scrollable section is implemented with max-height and overflow-y: auto. Buttons have hover effects using transition and transform: scale. The star rating uses JavaScript to toggle colors on hover.

Accessibility

Buttons include aria-label attributes, and the form uses semantic elements like <label> and <fieldset> for screen reader support.

Content Page

Technical Description

The Content Page uses internal linking with anchor tags (<a href="#section">) and a sticky sidebar for navigation (position: sticky). Images are styled with borders (border: 2px solid) and include alt text for accessibility.

Accessibility

Headings follow a logical hierarchy (<h1> to <h3>), and the sidebar navigation is keyboard-accessible with tabindex.

Challenges and Lessons Learned

Implementing the loader animation was challenging due to CSS timing issues, but I resolved it by adjusting the animation duration. I learned the importance of testing on multiple devices for responsiveness.

Compliance

My pages comply with Janet regulations by ensuring proper file naming (e.g., splash.html) and validating HTML/CSS with no errors. I also adhered to accessibility guidelines.
